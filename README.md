# ez-search

A simple cli search tool that was inspired by a section in the [De Morgan's laws](https://en.wikipedia.org/wiki/De_Morgan%27s_laws) Wikipedia article.

## Quick Start
```bash
mkdir build
cd build
cmake ..
make
```

## Command Line
```bash
ez-search INPUT SEARCH
```

## Running Tests
```bash
ctest --output-on-failure
```
